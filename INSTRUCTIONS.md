# 프로젝트 지침 (Project Instructions)

## 자동 Git 커밋 (Auto Git Commit)

이 프로젝트에서 코딩 작업을 수행하는 AI 에이전트는 앞으로 코드를 수정하거나 기능 변경을 완료할 때마다 반드시 다음 지침을 따라야 합니다:

1. **자동 커밋 진행**: 코드 수정이나 기능 추가를 성공적으로 구현하여 사용자에게 보고할 시점이 되면, 사용자가 명시적으로 요청하지 않더라도 자동으로 터미널을 사용해 커밋을 진행하세요.
2. **변경 사항 스테이징**: `git add .` 명령어로 변경 사항을 스테이징하세요.
3. **커밋 메시지 작성**: `git commit -m "작업 내용 요약"` 명령어를 실행하여 방금 수행한 작업의 핵심을 요약한 메시지로 커밋을 남기세요.
4. **결과 보고**: 커밋이 성공적으로 완료되었음을 사용자에게 함께 안내하세요.
